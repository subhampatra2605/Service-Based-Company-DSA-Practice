# 🧩 Container With Most Water

## 📥 Input

* `height = [1,8,6,2,5,4,8,3,7]`

---

## 📤 Output

* `49`

👉 Formed between:

```text id="cwmw-op1"
height[1] = 8
height[8] = 7
```

👉 Width:

```text id="cwmw-op2"
8 - 1 = 7
```

👉 Area:

```text id="cwmw-op3"
min(8,7) * 7 = 49
```

---

# 🔍 Pattern of the Problem

* **Two Pointers + Greedy**

👉 Core idea:

```text id="cwmw-core"
Area depends on:
min(height[left], height[right]) * width
```

---

# 🐢 Brute Force Approach

### Idea:

Check every possible pair

---

## 💻 Brute Force Code

```java id="cwmw-bf-code"
class Solution {
    public int maxArea(int[] height) {

        int maxWater = 0;

        // Try every pair
        for (int i = 0; i < height.length; i++) {

            for (int j = i + 1; j < height.length; j++) {

                // Calculate width
                int width = j - i;

                // Calculate height
                int h = Math.min(height[i], height[j]);

                // Calculate area
                int area = width * h;

                // Update maximum
                maxWater = Math.max(maxWater, area);
            }
        }

        return maxWater;
    }
}
```

---

## ⏱ Time Complexity

```text id="cwmw-tc1"
O(n²)
```

---

## ❌ Shortcomings

* Checks unnecessary pairs
* Too slow for large arrays

# ⚡ Optimal Solution (Two Pointers)

---

# 💻 Java Code (Line-by-Line Explanation)

```java id="cwmw-opt-code"
class Solution {
    public int maxArea(int[] height) {

        // Left pointer
        int left = 0;

        // Right pointer
        int right = height.length - 1;

        int maxWater = 0;

        // Continue until pointers meet
        while (left < right) {

            // Width between lines
            int width = right - left;

            // Smaller height determines water level
            int h = Math.min(height[left], height[right]);

            // Calculate current area
            int area = width * h;

            // Update maximum area
            maxWater = Math.max(maxWater, area);

            // Move smaller height pointer
            if (height[left] < height[right]) {

                left++;

            } else {

                right--;
            }
        }

        return maxWater;
    }
}
```

# 🎯 Why This Works

👉 Water depends on:

```text id="cwmw-proof1"
smaller height
```

👉 Moving larger height pointer is useless because:

```text id="cwmw-proof2"
width decreases
AND
smaller height still limits area
```

👉 So always move:

```text id="cwmw-proof3"
smaller height pointer
```

# ⚠️ Key Pitfalls

* ❌ Moving taller pointer
* ❌ Using max instead of min height
* ❌ Forgetting width calculation
