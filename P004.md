# 🧩 Subarray Sum Equals K

## 📥 Input

* `nums = [1,1,1]`
* `k = 2`

---

## 📤 Output

* `2`

👉 Valid subarrays:

```text id="ssk-op1"
[1,1]
[1,1]
```

---

# 🔍 Pattern of the Problem

* **Prefix Sum + HashMap**

👉 Core idea:

```text id="ssk-core"
If:
currentPrefixSum - previousPrefixSum = k

Then:
subarray sum = k
```

---

# 🐢 Brute Force Approach

### Idea:

Generate every subarray and calculate sum

---

## 💻 Brute Force Code

```java id="ssk-bf-code"
class Solution {
    public int subarraySum(int[] nums, int k) {

        int count = 0;

        // Start index
        for (int i = 0; i < nums.length; i++) {

            int sum = 0;

            // End index
            for (int j = i; j < nums.length; j++) {

                sum += nums[j];

                // Check if sum equals k
                if (sum == k) {
                    count++;
                }
            }
        }

        return count;
    }
}
```

---

## ⏱ Time Complexity

```text id="ssk-tc1"
O(n²)
```

---

## ❌ Shortcomings

* Recalculates subarray sums repeatedly
* Slow for large inputs

# ⚡ Optimal Solution (Prefix Sum + HashMap)

---

# 💻 Java Code (Line-by-Line Explanation)

```java id="ssk-opt-code"
import java.util.HashMap;

class Solution {
    public int subarraySum(int[] nums, int k) {

        // Map stores:
        // prefixSum -> frequency
        HashMap<Integer, Integer> map = new HashMap<>();

        // Important:
        // prefix sum 0 exists once initially
        map.put(0, 1);

        int prefixSum = 0;
        int count = 0;

        // Traverse array
        for (int num : nums) {

            // Update running prefix sum
            prefixSum += num;

            // Find required previous prefix sum
            int remove = prefixSum - k;

            // If exists, valid subarray found
            if (map.containsKey(remove)) {

                count += map.get(remove);

                // Add frequency because
                // multiple previous prefix sums may exist
            }

            // Store current prefix sum frequency
            map.put(prefixSum,
                    map.getOrDefault(prefixSum, 0) + 1);
        }

        return count;
    }
}
```

# 🎯 Why This Works

👉 Suppose:

```text id="ssk-proof1"
currentPrefixSum = x
```

We need:

```text id="ssk-proof2"
subarray sum = k
```

That means:

```text id="ssk-proof3"
x - previousPrefixSum = k
```

So:

```text id="ssk-proof4"
previousPrefixSum = x - k
```

👉 That’s exactly what we search in HashMap

# ⚠️ Key Pitfalls

* ❌ Forgetting `map.put(0,1)`
* ❌ Using sliding window (fails for negatives)
* ❌ Using `count++` instead of frequency addition
