# 🧩 Top K Frequent Elements

## 📥 Input

* `nums = [1,1,1,2,2,3]`
* `k = 2`

---

## 📤 Output

* `[1,2]`

👉 Frequency:

```text id="tkfe-op1"

2 → 2 times
3 → 1 time
```

👉 Top 2 frequent:

```text id="tkfe-op2"

```

# 🔍 Pattern of the Problem

* **HashMap + Heap (Priority Queue)**

👉 Core idea:

```text id="tkfe-core"
Count frequency
Then keep top k elements
```

---

# 🐢 Brute Force Approach

### Idea:

Count frequency → sort all elements by frequency

---

## 💻 Brute Force Code

```java id="tkfe-bf-code"
import java.util.*;

class Solution {
    public int[] topKFrequent(int[] nums, int k) {

        // Store frequencies
        HashMap<Integer, Integer> map = new HashMap<>();

        for (int num : nums) {
            map.put(num, map.getOrDefault(num, 0) + 1);
        }

        // Convert to list
        List<Integer> list = new ArrayList<>(map.keySet());

        // Sort by frequency descending
        Collections.sort(list,
            (a, b) -> map.get(b) - map.get(a));

        // Store top k
        int[] result = new int[k];

        for (int i = 0; i < k; i++) {
            result[i] = list.get(i);
        }

        return result;
    }
}
```

---

## ⏱ Time Complexity

```text id="tkfe-tc1"
O(n log n)
```

## ❌ Shortcomings

* Sorting entire list unnecessary
* We only need top k elements

# ⚡ Optimal Solution (Min Heap)

---

# 💻 Java Code (Line-by-Line Explanation)

```java id="tkfe-opt-code"
import java.util.*;

class Solution {
    public int[] topKFrequent(int[] nums, int k) {

        // Step 1: Count frequencies
        HashMap<Integer, Integer> freqMap = new HashMap<>();

        for (int num : nums) {

            freqMap.put(num,
                freqMap.getOrDefault(num, 0) + 1);
        }

        // Step 2: Create min heap
        // Smaller frequency comes first
        PriorityQueue<Integer> heap =
            new PriorityQueue<>(
                (a, b) -> freqMap.get(a) - freqMap.get(b)
            );

        // Step 3: Add elements into heap
        for (int num : freqMap.keySet()) {

            heap.add(num);

            // Keep only top k frequent elements
            if (heap.size() > k) {
                heap.poll();
            }
        }

        // Step 4: Store answer
        int[] result = new int[k];

        for (int i = k - 1; i >= 0; i--) {

            result[i] = heap.poll();
        }

        return result;
    }
}
```

# 🎯 Why This Works

👉 Min heap keeps:

```text id="tkfe-proof1"
smallest frequency at top
```

👉 Whenever heap exceeds k:

```text id="tkfe-proof2"
remove least frequent element
```

👉 Finally heap contains:

```text id="tkfe-proof3"
top k frequent elements
```
