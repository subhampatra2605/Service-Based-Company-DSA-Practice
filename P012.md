🧩 Search Insert Position

📥 Input

nums = [1,3,5,6]

target = 5



---

📤 Output

2


👉 Because:

nums[2] = 5


---

🔍 Pattern of the Problem

Binary Search


👉 Core idea:

If target not found,
return position where it should be inserted


---

🐢 Brute Force Approach

Idea:

Traverse array and find first position where:

nums[i] >= target


---

💻 Brute Force Code

class Solution {

    public int searchInsert(int[] nums, int target) {

        // Traverse array
        for (int i = 0; i < nums.length; i++) {

            // Correct insert position found
            if (nums[i] >= target) {
                return i;
            }
        }

        // Insert at end
        return nums.length;
    }
}


---

⏱ Time Complexity

O(n)


---

❌ Shortcomings

Linear traversal

Does not use sorted property



---

⚡ Optimal Solution (Binary Search)


---

💻 Java Code (Line-by-Line Explanation)

class Solution {

    public int searchInsert(int[] nums, int target) {

        // Left boundary
        int left = 0;

        // Right boundary
        int right = nums.length - 1;

        // Binary search
        while (left <= right) {

            // Find middle index
            int mid = left + (right - left) / 2;

            // Target found
            if (nums[mid] == target) {

                return mid;
            }

            // Search right half
            else if (nums[mid] < target) {

                left = mid + 1;
            }

            // Search left half
            else {

                right = mid - 1;
            }
        }

        // Left pointer becomes insert position
        return left;
    }
}


---

🔄 Complete Dry Run

Input:

nums = [1,3,5,6]
target = 2


---

Initial State

left = 0
right = 3


---

Step 1

Mid

mid = 1
nums[mid] = 3

3 > 2

👉 Search left side

Update:

right = 0


---

Step 2

left = 0
right = 0

Mid

mid = 0
nums[mid] = 1

1 < 2

👉 Search right side

Update:

left = 1


---

Loop Ends

left = 1
right = 0

Condition fails:

left > right


---

📤 Return

return left = 1

👉 Insert position:

[1, 2, 3, 5, 6]


---

🎯 Why This Works

👉 Binary search narrows search space

👉 If target not found:

left stops exactly at correct insert position


---

⏱ Complexity

Type	Value

Time	O(log n)
Space	O(1)



---

⚠️ Key Pitfalls

❌ Returning mid when target absent

❌ Wrong pointer updates

❌ Forgetting final return left
